Installing Apogeebrain
====================


The easiest way to install |apogeebrain| is using pip:

To get the very lastest version:

.. code-block:: bash

    pip install git+https://github.com/dnidever/apogeebrain



Dependencies
============

Here are the required packages.  Most of them can be pip installed.

- numpy
- scipy
- astropy
- matplotlib
- dlnpyutils
